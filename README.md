# SyntaxWear - E-commerce de Tênis

Bem-vindo ao repositório do **SyntaxWear**, um projeto de e-commerce moderno focado na venda de calçados de alta qualidade. Este projeto foi desenvolvido com foco em uma interface limpa, responsiva e com uma experiência de usuário fluida.

## 🚀 Sobre o Projeto

O SyntaxWear é uma landing page de um e-commerce de sneakers que apresenta coleções divididas por categorias (Casual, Desportivo, Moderno e Futurista). O objetivo principal é demonstrar as habilidades de front-end, utilizando técnicas modernas de CSS e organização de código.

## ✨ Funcionalidades

- **Design Responsivo:** O site se adapta a diferentes tamanhos de tela (Mobile e Desktop).
- **Menu Hamburguer:** Navegação otimizada para dispositivos móveis utilizando apenas HTML e CSS.
- **Seções de Destaque (Hero):** Banner principal com chamadas para ação (CTA).
- **Categorias Interativas:** Cards que direcionam para diferentes nichos de produtos.
- **Grid de Produtos:** Layout organizado para exibição de sneakers e modelos.
- **Newsletter:** Área para captura de e-mails no rodapé.
- **Redes Sociais:** Links integrados para as principais plataformas sociais.

## 🛠️ Tecnologias Utilizadas

Para a construção deste projeto, foram utilizadas as seguintes ferramentas:

- **HTML5:** Estruturação semântica do conteúdo.
- **CSS3:** Estilização modularizada, utilizando:
  - Flexbox e CSS Grid para layouts.
  - Variáveis CSS para facilitar a manutenção de cores e fontes.
  - Importação de fontes externas (Google Fonts: Ubuntu e Roboto Mono).
- **SVG:** Ícones e logo de alta qualidade e escaláveis.

## 🚀 Como Executar o Projeto

1. Faça o clone deste repositório ou baixe os arquivos.
2. Navegue até a pasta raiz do projeto.
3. Abra o arquivo `index.html` em qualquer navegador moderno.
   - *Dica: Se estiver usando o VS Code, utilize a extensão "Live Server" para uma melhor experiência.*
